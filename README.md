# Bank Customer Churn Analysis using Power BI

## Project Overview

This project analyzes customer churn behavior in a banking environment using Power BI.

The dashboard provides insights into customer retention, customer exits, demographics, geography, credit card ownership, and customer activity patterns.

The objective is to identify factors contributing to customer churn and support data-driven business decisions.

## Tools Used

---

## Dashboard Features

### Dashboard Page 1

- KPI Cards
  - Total Customers
  - Active Customers
  - Exit Customers
  - Retained Customers
  - Credit Card Holders
  - Inactive Customers

### Analysis Performed

- Exit Customers by Month
- Customer Category Trends
- Exit Customers by Gender
- Exit Customers by Credit Card Category
- Smart Narrative Insights

### Dashboard Page 2

- Monthly Churn Percentage Analysis
- Geography-wise Customer Distribution
- Customer Retention by Credit Type

---

## Key Insights

- Customer churn fluctuates across different months.
- Female customers represent a slightly higher percentage of exit customers.
- Credit card holders contribute significantly to the customer base.
- Geography influences customer distribution.
- Customer retention varies across different credit score categories.
